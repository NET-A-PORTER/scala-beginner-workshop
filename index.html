vals and vars

  • val (value)
    • Once assigned cannot be reassigned
  • var (variable)
    • Can be reassigned
  • Immutability first
    • Use val by default
    • Only use var when necessary

Some Solutions

def asString = 
  stars + "* " + 
  comment.map("[comment:" + _ + "]").
  getOrElse("[no comment]")

def asString = 
  stars + "* " + (
  if(comment.nonEmpty) "[comment:" + comment.get + "]" 
  else "[no comment]")

  case Some(c) => s"[comment:$c]" 
  case None => "[no comment]"
